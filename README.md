# Tic Tac Toe Game

A simple Tic Tac Toe game built using HTML, CSS, and JavaScript as part of my JavaScript learning journey. This project helped me understand core concepts like DOM manipulation, event handling, and basic game logic.

# Features
🧑‍🤝‍🧑 Two-player game (Player X vs Player O)
🔄 Turn-based gameplay
✅ Win detection logic
🔁 Restart / Reset game option
🎯 Simple and clean UI
# Tech Stack
HTML – Structure of the game
CSS – Styling and layout
JavaScript – Game logic and interactivity
# What I Learned
DOM manipulation in JavaScript
Handling user events (clicks)
Implementing game logic
Managing game state
Writing cleaner and structured code
